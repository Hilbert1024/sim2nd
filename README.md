# sim2nd
Graph embedding framework based on random walk. 
This is a beta version 0.1.0 released on 2020-02-14.

# Requirements
+ numpy
+ networkx
+ gensim
+ scipy
+ scikit_learn

# Start
```markdown
cd sim2nd
pip install -r requirements.txt
cd src
python main.py
```
